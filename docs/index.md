# nqxpack

:::{nkx-hero}

:image: _static/hero-placeholder.svg

A simple, human-inspectable `.nk` format for saving and loading JAX-based ML objects —
Flax models, NetKet variational states, and plain NumPy data. One command to save,
one command to load.
